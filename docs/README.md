## How to use

1. Open `01_PROJECT_REPORT.md` — fill certificate/declaration names and dates.
2. Open `02_GEMINI_DIAGRAM_PROMPTS.md` — copy each prompt into **Google Gemini** to generate figures.
3. Insert generated images into your Word/PDF report as Fig 4.1, 4.2, etc.
4. Use `03_SRS_SHORT.md` for viva or quick reference.

## Note on reference PDF

The file `JAVA & SPRING BOOT_compressed.pdf` in the project root is a **scanned/image PDF** (text could not be extracted automatically). These documents follow the **same typical sections** used in that report format: Introduction, Analysis, Design, Implementation, Testing, Conclusion, and UML diagrams.
